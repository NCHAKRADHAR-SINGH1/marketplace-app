## 🚀 Quick Start (3 Steps)

### Step 1: Start Backend
```bash
cd backend
npm install
npm run seed
npm run dev
```
✅ Backend running on `http://localhost:5000`

### Step 2: Start Web App
```bash
cd ../web
npm install
npm run dev
```
✅ Web app running on `http://localhost:5173`
